# Castlemania
This is an unlicensed Freeware Castlevania fangame where you can play as a vampire hunter (Eric Lecarde) or Dracula himself. This project is not affiliated in any way with KONAMI, the current owners of Castlevania franchise.

### **For more info visit:**
https://acytra.itch.io/castlemania
